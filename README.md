# ImplementAI-2018
Repo for ImplementAI project


To run the frontend:
    * For Windows:
        * set FLASK_APP=ImplementAI-2018
    * For LInux:
        * export FLASK_APP=ImplementAI-2018
    *For both:
        * pip install -e .
        * flask run
        * access the frontend at http://127.0.0.1:5000
